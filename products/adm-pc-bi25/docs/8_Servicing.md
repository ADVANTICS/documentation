# **8\. Servicing**

## 

## 8.1. Service & repair procedures

In case of any technical questions or issues, we recommend creating a support ticket on [ADVANTICS Helpdesk](https://advantics.atlassian.net/servicedesk/customer/portals). Your ticket will be assigned to the appropriate engineer for your assistance.

## 8.2. Firmware or other updates

Over time updates for the BI25 module may become available. These updates may improve existing operating modes or add new operating modes or features for specific situations. It is recommended to only update the firmware if there is a specific need to do so. To update the firmware, ADVANTICS Flash Programming Utility (AFPU) application is executed on a PC that is connected to the CAN bus. For detailed instructions, please see the AFPU guide.
